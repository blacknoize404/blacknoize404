## 📚 Academic Resources
- 📖 **[Learn HTTP Protocol & Software Architecture with MiniJWS](https://github.com/blacknoize404/MiniJWS/tree/master/docs)**: MiniJWS isn't just a functional server; it was built explicitly as an educational initiative. In its extensive documentation, beginners and students can learn:
  - How the HTTP/1.1 standard works from scratch.
  - Software design decisions, including why specific Java classes and design patterns are used.
  - How to implement thread-pool concurrency.
  - How core web features like routing, middlewares (CORS, Gzip, Rate Limit), and static file serving are built without relying on heavy third-party frameworks. 
  It is an excellent resource for seeing university-level networking and programming concepts applied in a real-world project.
